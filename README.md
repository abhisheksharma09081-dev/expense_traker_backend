# Expense Tracker Backend

A RESTful Expense Tracker API built with Spring Boot.

## Tech Stack
- Java 21
- Spring Boot 3
- Spring Security
- JWT Authentication
- MongoDB Atlas
- Docker
- Swagger/OpenAPI
- JUnit 5
- Mockito

## Features
- User Registration & Login
- JWT Authentication
- Income Management
- Expense Management
- Category Management
- Pagination & Sorting
- Input Validation
- Global Exception Handling
